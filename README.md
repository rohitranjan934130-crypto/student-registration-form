🎓 Student Registration Form

A simple web-based application to **add and display student details** using HTML, CSS, and JavaScript.


## Features

* Add student details (Name, Age, Course)
* Display data in a table
* Input validation (no empty fields)
* Clean and responsive UI
* Instant data display (no page reload)


##  Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

##  Project Structure

```
student-registration-form/
│── index.html
```

---

##  How to Run the Project

1. Download or clone the repository
2. Open the folder
3. Double-click on `index.html`
4. The project will open in your browser

---

##  How It Works

* User enters student details in the form
* Clicks **"Add Student"**
* JavaScript captures input data
* Data is inserted into the table dynamically
* Form fields reset after submission

---

##  Limitations

* Data is not saved permanently (refresh = data lost)
* No backend/database integration
* Basic validation only

---
